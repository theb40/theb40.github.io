COVID 19 Data exploration

Introduction

This project was created for practicing and demonstrating data exploration techniques with SQL using Azure Data Studio. Originally inspired by Alex’s Data Analyst Bootcamp, I took advantage of my previous experience with Google’s Big Query to explore a new database management tool.

*Note that the Data Analyst Bootcamp was more of refresher for me than a fresh start, so I didn’t follow Alex’s instruction but rather took the assignment as a stand-alone personal project.

Data

The data used in this project is open-sourced and publicly available at OurWorldInData.org.

Citation: Edouard Mathieu, Hannah Ritchie, Lucas Rodés-Guirao, Cameron Appel, Charlie Giattino, Joe Hasell, Bobbie Macdonald, Saloni Dattani, Diana Beltekian, Esteban Ortiz-Ospina and Max Roser (2020) - "Coronavirus Pandemic (COVID-19)". Published online at OurWorldInData.org. Retrieved from: 'https://ourworldindata.org/coronavirus' [Online Resource]

	location,
	continent,
	MAX(total_cases) as recorded_cases,
	MAX(total_deaths) as confirmed_deaths,
	FORMAT(MAX(total_deaths) /MAX(total_cases), 'P') as mortality_rate
FROM
	COVID19..CovidDeaths
GROUP BY
	location,
	continent
HAVING
	continent is not NULL
ORDER BY
	recorded_cases DESC

Percentage of affected population


-- a subquery to clear out world and continents rows 
WITH temp_covid_data as ( 
SELECT 
  *
FROM 
  COVID19.dbo.CovidDeaths
WHERE   
  continent is not NULL
)
SELECT 
  location,
  MAX(population) population,
  MAX(total_cases) recorded_cases,
  FORMAT(MAX(total_cases)/MAX(population), 'P') as affected_percentage
FROM 
  temp_covid_data
GROUP BY 
  population,
  location
HAVING 
  population is not NULL 
  AND 
  MAX(total_cases) is not NULL
ORDER BY 
  MAX(total_cases)/MAX(population) DESC
-- the FORMAT clause changed the datatype of the column into nvarchar

We can also combine these queries to make a comprehensive table about affected percentage and mortality rate. Let’s pretend that I wanted to get all of these information but specifically about Asia: 

WITH temp_covid_data as ( 
SELECT 
    *
FROM 
    COVID19.dbo.CovidDeaths
WHERE   
    continent is not NULL
    AND 
    continent = 'Asia'
)
SELECT 
    location,
    MAX(population) population,
    MAX(total_cases) recorded_cases,
    MAX(total_deaths) confirmed_deaths,
    FORMAT(MAX(total_cases)/MAX(population), 'P') affected_percentage,
    FORMAT(MAX(total_deaths) /MAX(total_cases), 'P') mortality_rate
FROM 
    temp_covid_data
GROUP BY 
    population,
    location
HAVING 
    population is not NULL 
    AND 
    MAX(total_cases) is not NULL
ORDER BY 
    population DESC

Vaccination rate

SELECT 
  cvd.location,
  MAX(population) population,
  MAX(people_vaccinated) vaccinated,
  FORMAT(CAST(MAX(people_vaccinated) as float)/MAX(population), 'P')
	  as vaccine_coverage_rate
-- added a column to calculate the vaccinated percentage
FROM 
  COVID19..CovidDeaths cvd 
  LEFT JOIN
      COVID19..CovidVaccinations cvv
  ON 
      cvd.location = cvv.location
GROUP BY 
  cvd.location
ORDER BY 
  CAST(MAX(people_vaccinated) as float)/MAX(population) DESC

Vaccination rate vs Mortality rate 


 WITH temp_data as (
    SELECT
        cvd.location as location,
        cvd.continent as continent,
        cvd.population as population,
        cvd.total_cases as total_cases,
        cvd.total_deaths as total_deaths,
        cvv.people_vaccinated as vaccinated
    FROM 
        COVID19..CovidDeaths cvd 
        LEFT JOIN
            COVID19..CovidVaccinations cvv
        ON 
            cvd.location = cvv.location
    WHERE 
        cvd.continent is not NULL
        AND 
        cvv.people_vaccinated is not NULL
 )
 SELECT
    location,
    continent,
    MAX(population) population,
    MAX(vaccinated) vaccinated,
    ROUND(MAX(total_cases)/MAx(population) * 100, 2) 
	    as affected_rate,
    ROUND(CAST(MAX(vaccinated) as float)/MAX(population)*100, 2) 
	    as vaccine_coverage_rate,
    ROUND(MAX(total_deaths)/MAX(total_cases)*100, 2)
	    as mortality_rate
 FROM 
    temp_data
 GROUP BY 
    location,
    continent
 HAVING 
    MAX(total_deaths)/MAX(total_cases) is not NULL
 ORDER BY 
    ROUND(CAST(MAX(vaccinated) as float)/MAX(population)*100, 2)
